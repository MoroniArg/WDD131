Overview

Purpose

The purpose of this website will be to inform Pokemon Go players about upcoming global events, tips and tricks they should fallow to get full advantage of the events, and allow them to get to know a little more about our community. The site would have information about how to access the Discord and Campfire groups and informations about our usual meetup places for events, including some pictures of past events.

Audience

The intended audience for the website will mainly be the Pokemon Go Community here in Rexbug, Idaho, and potentially cities around.

Dynamic elements

  • Countdown timer displaying the next event when we will meet up.
  • List of upcoming meetup events containing the title, date, location, and description.
  • Conditional branching to check if there is or there is no global non-meetup events currently.
  • A dark mode toggle to allow to switch between light and dark themes.
